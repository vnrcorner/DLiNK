## 🛠️ Setup

1. Clone the repository
```bash
git clone https://github.com/vnrcorner/dlink.git
cd dlink
```

2. Configure environment variables
```bash
cp .env.example .env
```

3. Run with PHP server
```bash
php -S localhost:8000 -t public
```

## 💻 Tech Stack

- PHP 7.4+
- JavaScript (ES6+)
- CSS3 with Custom Properties
- Socket Programming

## 📋 Requirements

- PHP 7.4 or higher
- PHP Sockets Extension
- Modern web browser
